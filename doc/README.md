# Documentation

This folder contains project-level references and visuals for the **Customer Risk Profiling System**. These materials help explain the system vision, architecture, and data model before exploring the implementation details.

⚠️ Note:
This is an **ongoing academic learning project** developed using a microservices architecture. The documentation, system design, and implementation details may change over time based on:

- evolving client-style requirements
- architecture improvements
- feature enhancements
- refactoring decisions
- learning progress during development

Because this project is actively under development, some inconsistencies or mistakes may exist and will be improved continuously as the system matures.
